#Loan Approval Predictin using Machine Learning
#Project Overview
This project predicts loan approval status using Logistic Regression model.
The model is trained and evaluated on the 'loan_approval_1000.csv'dataset.
** Model Perforance**
-Accuaracy : 79%
-F1.Score  :0.79
-Class Precision /Recall :0.83 / 0.87
